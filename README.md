# Badminton Club Portal

## Overview
A secure, automated portal for the Canton High School Badminton Club.  
Features membership management, event registration, announcements, and admin controls — built using Google Workspace tools.

## Features
- Google login authentication (school accounts only)
- Member database (Google Sheets)
- Event registration system (Google Form → Sheets)
- Admin panel with role-based access control
- Live announcements and events on Google Sites
- Automatic confirmation emails (Apps Script)

## Technologies Used
- Google Sites (frontend)
- Google Forms (input)
- Google Sheets (database)
- Google Apps Script (backend automation)

## Security
- Role-based access control (Admin vs Member)  
- Page-level permissions on Admin Panel  
- School-domain restriction
